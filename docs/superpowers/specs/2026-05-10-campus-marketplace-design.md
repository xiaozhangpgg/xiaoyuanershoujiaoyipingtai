# 校园二手交易平台设计文档

## 项目概述

**项目定位：** 毕业设计项目，面向本校学生的二手交易平台

**技术栈：**
- 前端：Next.js 14+ (App Router) + Tailwind CSS + shadcn/ui
- 后端：Next.js API Routes
- 数据库：PostgreSQL + Prisma ORM
- 认证：NextAuth.js (邮箱/密码)
- 实时消息：Socket.io
- 文件存储：本地 `public/uploads/`

**适配：** 仅移动端（375px 宽度）

---

## 核心功能

### 1. 用户系统

**注册流程：**
- 填写邮箱、密码、昵称、学号
- 上传学信网照片
- 系统提取学信网中的学号，与填写的学号比对
- 匹配成功则认证通过，仅本校生可注册

**登录：** 邮箱 + 密码登录

**个人中心：**
- 头像、昵称、认证状态
- 我发布的商品
- 我的收藏

### 2. 商品管理

**发布商品：**
- 标题
- 描述
- 价格
- 分类（教材、数码、生活用品、服饰、其他）
- 新旧程度（全新、几乎全新、轻微使用痕迹、明显使用痕迹）
- 物品所在位置（如：南区6号楼、图书馆附近）
- 图片上传（最多9张）

**商品状态：** 在售、已售、已下架

**操作：** 编辑、删除、上下架

### 3. 搜索与筛选

- 关键词搜索（标题、描述）
- 分类筛选（横向滚动标签）
- 价格区间筛选
- 地址筛选
- 按发布时间排序

### 4. 即时消息

**触发：** 商品详情页点击"联系卖家"

**消息列表页：**
- 显示最近对话列表
- 每项显示：对方头像、昵称、最后消息、时间

**聊天页：**
- 全屏聊天界面
- 消息气泡（左对方、右自己）
- 底部固定输入框 + 发送按钮
- WebSocket 实时推送新消息

## 数据库设计

### User 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Int | 主键 |
| email | String | 邮箱（唯一） |
| password | String | 密码（bcrypt 加密） |
| nickname | String | 昵称 |
| avatar | String | 头像路径 |
| studentId | String | 学号 |
| verified | Boolean | 是否已认证 |
| createdAt | DateTime | 注册时间 |

### Product 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Int | 主键 |
| title | String | 标题 |
| description | String | 描述 |
| price | Float | 价格 |
| images | String[] | 图片路径数组 |
| categoryId | Int | 分类 ID |
| condition | Enum | 新旧程度 |
| location | String | 物品所在位置 |
| status | Enum | 在售、已售、已下架 |
| sellerId | Int | 卖家 ID |
| createdAt | DateTime | 发布时间 |

### Category 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Int | 主键 |
| name | String | 分类名称 |

**预设分类：** 教材、数码、生活用品、服饰、其他

### Message 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Int | 主键 |
| content | String | 消息内容 |
| senderId | Int | 发送者 ID |
| receiverId | Int | 接收者 ID |
| productId | Int | 关联商品 ID |
| isRead | Boolean | 是否已读 |
| createdAt | DateTime | 发送时间 |

### Conversation 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Int | 主键 |
| user1Id | Int | 用户 A |
| user2Id | Int | 用户 B |
| productId | Int | 关联商品 |
| lastMessageAt | DateTime | 最后消息时间 |

### Transaction 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | Int | 主键 |
| productId | Int | 商品 ID |
| sellerId | Int | 卖家 ID |
| buyerId | Int | 买家 ID |
| price | Float | 成交价格 |
| status | Enum | PENDING / SELLER_CONFIRMED / COMPLETED / CANCELLED |
| sellerConfirmedAt | DateTime | 卖家确认时间 |
| buyerConfirmedAt | DateTime | 买家确认时间 |
| createdAt | DateTime | 创建时间 |

**交易状态流转：**
```
PENDING（卖家发起）
    ↓
SELLER_CONFIRMED（卖家标记已售，选择买家）
    ↓
COMPLETED（买家确认交易完成）
```

可选：任意一方取消 → CANCELLED

## 设计规范

### 色彩方案

| 角色 | 色值 | 用途 |
|------|------|------|
| Primary | #22C55E | 主色调、按钮、高亮 |
| Primary Dark | #16A34A | hover 状态、深色文字 |
| Primary Light | #F0FDF4 | 浅色背景、标签底色 |
| CTA | #F97316 | 强调操作（如价格） |
| Text | #0F172A | 主文字色 |
| Muted | #64748B | 辅助文字、时间戳 |
| Border | #E2E8F0 | 分割线、卡片边框 |

### 字体

使用系统字体栈，中文优先渲染：
```
-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif
```

### 图标

**图标库：** Lucide（与 shadcn/ui 原生集成）

```bash
npm install lucide-react
```

```tsx
import { Home, Plus, MessageCircle, User, Search, BookOpen, Laptop, Shirt, Headphones } from 'lucide-react'
```

**规范：**
- 使用 SVG 矢量图标，禁止使用 emoji 作为 UI 图标
- 统一 viewBox="0 0 24 24"，尺寸 w-5 h-5 或 w-6 h-6
- 底部导航图标 22px，内联图标 16px

### 圆角

| 场景 | 类名 | 像素 |
|------|------|------|
| 卡片/按钮 | rounded-lg | 8px |
| 大卡片/弹窗 | rounded-2xl | 16px |
| 标签/头像 | rounded-full | 9999px |
| 输入框 | rounded-full | 9999px |

### 间距

使用 Tailwind 默认间距系统，关键间距：
- 页面内边距：px-4 (16px)
- 卡片间距：gap-2 (8px)
- 区块间距：py-3 (12px)

### 交互规范

- touch target 最小 44x44px
- 相邻 touch target 间距 ≥ 8px
- hover/active 状态使用 transition-colors duration-200
- 所有可点击元素添加 cursor-pointer
- 加载中状态：禁用按钮 + spinner
- prefers-reduced-motion 媒体查询需尊重

### 适配

- 仅移动端，宽度 375px
- 使用 `<meta name="viewport" content="width=device-width, initial-scale=1">`
- 最小字号 13px，正文 14-15px
